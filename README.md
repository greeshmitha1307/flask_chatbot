\# Simple Flask Chatbot



A beginner-friendly chatbot built using \*\*Python Flask\*\* and a clean web UI.



\## 🚀 Features

\- Rule-based chatbot responses

\- Clean chat UI

\- Sends messages without reloading (AJAX fetch)

\- Python backend + HTML frontend

\## 🛠 How to Run

1\. Install dependencies:

pip install flask



2\. Run the app:

python app.py



3.Open browser:



http://127.0.0.1:5000



Your chatbot is ready.





\## 🧪 Example Questions



You can try asking the chatbot:



\- "hello"

\- "how are you"

\- "what is your name"

\- "who created you"

\- "tell me a joke"

\- "what's the time"

\- "what's today's date"

\- "what is machine learning"

\- "why is the sky blue"

\- "define blockchain"

\- "explain artificial intelligence"



The bot responds directly to simple questions,  

and for more complex ones it provides a Google search link.





🧠 How They Respond



✔ Simple questions → direct answers

✔ Complex questions → Google search link

✔ Jokes → fun response

✔ Time / date → current system time
